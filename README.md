# Eucator Passagens

## Descricao do projeto

O Eucator Passagens e uma aplicacao web para consulta, selecao e compra de passagens de onibus. O sistema permite buscar viagens, escolher assentos, realizar login, completar cadastro de cliente, reservar poltronas temporariamente e finalizar compras com pagamento via Pix.

## Objetivo da aplicacao

O objetivo da aplicacao e facilitar a venda de passagens rodoviarias, oferecendo uma experiencia integrada para o cliente selecionar uma viagem, reservar assentos, acompanhar a disponibilidade em tempo real e concluir o pagamento de forma simples.
